# 🌟 Horitoki Saree Digital Catalogue

[![Live Demo](https://img.shields.io/badge/demo-online-brightgreen.svg)](https://catalog.horitoki.com)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

An interactive, high-performance digital flipbook designed for the **Horitoki** premium saree collection. This catalogue provides a tactile "real-book" experience for customers to browse through saree designs with ease on any device.

## ✨ Key Features

* 📖 **3D Flip Experience:** Powered by Turn.js for a realistic page-turning animation.
* 🔍 **Advanced Pan & Zoom:** Users can click the zoom icon and drag/pan the image to inspect intricate fabric details and embroidery.
* 🛒 **Instant Order Integration:** Dedicated WhatsApp and Messenger buttons for every saree, pre-filled with the specific product number.
* 🛡️ **Image Protection:** Implemented "Anti-Save" layers and right-click restrictions to prevent direct image downloads (while allowing screenshots).
* ⚡ **Ultra-Fast Performance:** Optimized loop logic and Lazy Loading to handle 140+ high-resolution PNG/JPG images without lagging.
* 🎵 **Audio Feedback:** Realistic sound effects upon page turning for an immersive experience.
* 📱 **Fully Responsive:** Adapts perfectly to mobile, tablet, and desktop viewports.

## 🚀 Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Libraries:** [jQuery](https://jquery.com/), [Turn.js](http://www.turnjs.com/), [Lucide Icons](https://lucide.dev/)
- **Hosting:** [Netlify](https://www.netlify.com/)
- **Audio:** Custom flip sound-effect (MP3)
 **Run:** Open `index.html` in any modern web browser.
